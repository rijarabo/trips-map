About this map

This map shows the 738 citibike stations and where each of the 73,728 trips started or ended by each hour of January 22, 2018.

About the icons:

To see the trips taken each hour displayed sequentially.
To pause the animation anytime.
To stop and restart the animation.
To see some graphics about citibike trips.
To show or hide the layer control.
To see this message again.

Finally, by clicking any station, you can find out more information about the trips taken at any given hour.

About the colors:

At first all the points will be displaying location of citibike stations in blue.

When selecting layers, or playing the animation, the stations will change color and size to display different properties:

If a station had more incoming trips during the hour, the station will be green.
If a station had more outgoing trips during the hour, the station will be red.
The size of the points depends on how many trips where taken from each station any given hour.

(All of this is also explained in the legend)
